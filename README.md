# 项目简介

## 核心功能
- 基于YOLOv8的水面目标检测，支持5类目标识别：Water_pollution、Floating_matter、Abandoned_ships、Fishing_and_Aquaculture、Waste
- 支持多种输入方式：单张图片、图片文件夹、视频文件、本地摄像头
- 提供完整UI界面，可实时显示目标位置、目标总数和置信度信息
- 支持自定义检测参数：置信度阈值、IOU阈值
- 支持选择显示标签和原图
- 支持批量检测功能，可查看每一张检测结果
- 包含2500张标注数据集和训练好的模型

## 适用场景
- 水面环境污染监测
- 水面漂浮物检测
- 船只监控与安全管理
- 渔业和养殖区域监测
- 水面垃圾识别与管理

## 技术特点
- 使用Pycharm/VSCode+Anaconda开发环境
- 基于Python 3.9，依赖opencv-python、PyQt5、ultralytics、torch1.9等技术栈
- 提供完整的源代码、UI界面源文件、数据集和项目运行教程
- 配置成功环境后可直接使用，无需额外训练
